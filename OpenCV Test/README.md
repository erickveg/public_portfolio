# Overview

This folder contains innovative software solutions to real-world problems designed and developed by Erick Vega.
Each project has been clearly documented and contains ideas to future improvements. The purpose of this folder is to show OpenCV features and capabilites combined with Anaconda which is a machine learning module.

Each section of this software has been written in Visual Studio Code taking advantage of it libraries and extensions.

Required libraries:
* OpenCV
* Anaconda 

Python has been mainly used to develop this software.
